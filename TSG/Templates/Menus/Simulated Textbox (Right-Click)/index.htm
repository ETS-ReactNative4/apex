The menu is optimized for screen reader and keyboard only users:

  • Set focus to the edit field.
  • Activate the menu using the keyboard by pressing Shift+F10 or the Applications key.
  • Press the Up and Down Arrow keys to move focus between the menu options.
  • Press the Left and Right Arrow keys to open or close submenus.
  • Press any letter or number to jump to a menu item starting with that character or digit.
  • Press Enter to activate a menu option.
  • Press Escape to close the currently open menu.
  • Press Tab or set focus back on the triggering element to close the menu and all submenus.
  • Mouse users can right-click the edit field and click the desired menu option as usual.
